## Localhost setup

Lag .env-fil med innhold:

```
ACCESS_TOKEN="Sett inn Instagram access token"
```

Installer packages:

```
npm install
```

Serve:

```
npm run dev
```

Gå til:

[localhost:5000](http://localhost:5000/)
